# Markdown标记
>`#`

一级标题，以此类推，共六级；

>`*` 或 `-`

无序列表；

>`1.`

数字加"."为有序列表；

>`[](...链接地址...)`

插入链接

>`![](...连接地址...)`

插入图片

>`*ababab*`

一对儿*紧贴文字包裹为斜体；

>`**adfasdfsd**`

两对儿*紧贴文字包裹为粗体；

>`|&-`

表格直接用竖线和短横线分割;

>``

代码用反引号包裹；

>tab

tab键缩进

>`***`

分割线
